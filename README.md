**Mediq** is a state-of-the-art Clinical Decision Support System (CDSS) designed for medical knowledge retrieval, advanced cognitive reasoning, and real-time clinical assistant triage. Built for high-reliability environments, Mediq provides verified healthcare professionals with safe, rapid, and structured clinical insights.

---

## 🌟 Key Capabilities

*   **Integrated Clinical Decision Support**: Streamlines provider diagnostic workflows, triage prioritization, and medical inquiry parsing.
*   **Unified Medical Reference Ingestion**: Consolidates medical data and alerts from verified international healthcare databases, regulatory frameworks, and peer-reviewed journals.
*   **Secure Multi-Agent Reasoning**: Orchestrates cognitive evaluations to cross-reference differentials, guidelines, and patient parameters.
*   **High-Fidelity Semantic Retrieval**: Features robust search capabilities to rapidly extract key clinical evidence across extensive medical libraries.

---

## ⚙️ Development Setup

### Installation
```bash
git clone https://github.com/king007pin/Mediq.git
cd Mediq
npm install
```

### Run Server
```bash
npm run dev
```
